# 4 Servo Sweep Project

This is a simple Arduino project that demonstrates how to control 4 servo motors simultaneously based on the basic Sweep example.

## Description
The project initializes 4 servo motors to perform a sweeping motion. After 2 seconds of operation, all servos stop and hold their position at a 90-degree angle.

## Components
* Arduino Uno Board
* 4 Servo Motors 
* Jumper Wires
* Breadboard 

## Circuit Diagram
Please refer to the `Circuit Diagram.png` file included in this repository for the wiring connections.

## How to Use
1. Open the `Arduino code` file in the Arduino IDE.
2. Connect your 4 servos to the designated PWM pins specified in the code.
3. Upload the code to your Arduino board.
